## Prerequisites

### Ubuntu 14.10

```bash
$ sudo apt-get install build-essential clang llvm-dev libboost-all-dev
```

## Build

```bash
$ ./waf configure
$ ./waf build -p
```

## Test

```bash
$ ./waf test
```
