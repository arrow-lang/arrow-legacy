# Arrow
[![Build Status](https://travis-ci.org/arrowlang/arrow.svg?branch=master)](https://travis-ci.org/arrowlang/arrow)

## Prerequisites

### Ubuntu 14.10

```bash
$ sudo apt-get install build-essential clang llvm-dev libboost-all-dev libgmp-dev
```

## Build

```bash
$ ./waf configure
$ ./waf build -p
```

## Test

```bash
$ ./waf test
```
